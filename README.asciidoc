== Controls

//.Controls
[width="30%"]
,===
Button, Effect
Arrows or WASD, move (down arrow to stop moving)

Space, jump

k, kill player
r, respawn
c, clear particles
f, boost dash
q, quit

right click, respawn player at cursor
left click, place blocks
,===
